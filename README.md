## 🚀 Quick Start

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone or download the project**
   ```bash
   # If using git
   git clone https://github.com/arslan-corporation/website.git
   cd website
   
   # Or simply download and extract the files
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   - Navigate to `http://localhost:3000`
   - The website will automatically open in your default browser

### Alternative Setup (Without Node.js)

If you don't have Node.js installed, you can simply:

1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. The website will work perfectly as a static site

## 🔧 Customization

### Adding New Projects
1. Open `index.html`
2. Find the projects section
3. Add a new project card following the existing pattern
4. Update the project count in the stats section

### Changing Colors
1. Open `styles.css`
2. Find the CSS custom properties at the top
3. Update the color values as needed

### Updating Company Information
1. Edit contact details in the contact section
2. Update social media links in the footer
3. Modify company description in the about section
